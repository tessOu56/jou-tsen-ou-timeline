# Jou-Tsen Ou｜Academic Record（學術歷程時間軸）

展示歐柔岑（Jou-Tsen Ou）學術歷程的靜態網頁：論文、預印本、里程碑、學經歷。
純 HTML/JS、無需建置，瀏覽器直接開 `index.html` 即可瀏覽。

## 功能

- 關鍵字搜尋（標題／作者／期刊／標籤，含高亮）
- 類別與驗證狀態篩選；點標籤快速搜尋
- 每筆紀錄附公開來源連結
- 目前篩選結果可匯出 JSON

## 資料

紀錄集中於 `data/records.js`，每筆皆有公開來源；標示「待確認」者代表尚未取得公開佐證。
